# 用户头像和昵称集成 - 功能说明

## 功能概述

✅ 已完成用户头像和昵称显示功能：

- **登录前**：显示"登录"按钮
- **登录后**：显示用户头像和昵称，点击可打开下拉菜单
- **下拉菜单**：包含个人资料、设置、登出等选项

## 实现细节

### 新结构

```
src/components/common/
└── UserProfile.vue  (新组件 - 用户信息展示或登录按钮)

src/components/layout/
└── Header.vue  (已更新 - 使用 UserProfile)
```

### 工作流程

```
用户登录
  ↓
Token 和用户信息保存到 Pinia store
  ↓
Header 中的 UserProfile 组件自动检测登录状态
  ↓
显示用户头像和昵称（替换登录按钮）
  ↓
点击头像可打开下拉菜单
  ↓
菜单中可选择登出
```

### 手机版

- 只显示头像（昵称隐藏节省空间）
- 相同的下拉菜单功能

## 功能特点

✨ **头像显示**

- 使用用户返回的头像 URL
- 如果没有头像，显示昵称首字母 + 渐变背景

✨ **点击交互**

- 点击头像切换下拉菜单
- 点击菜单项后自动关闭菜单
- 点击页面其他地方也会关闭菜单

✨ **登出功能**

- 点击"登出"按钮清除所有登录数据
- 跳转回登录页面

✨ **响应式设计**

- PC 端：显示头像和昵称
- 手机端：只显示头像（优化空间）

## 数据来源

```typescript
userStore.user = {
  id: 1, // 用户 ID
  nickname: "昵称", // 显示的昵称
  image: "头像URL", // 头像图片
  role: 0, // 用户角色
  status: 1, // 用户状态
};
```

## 可定制选项

### 1. 菜单项

在 `UserProfile.vue` 中修改：

```vue
<!-- 添加新菜单项 -->
<button
  @click="
    () => {
      /* 自定义行为 */
    }
  "
>
  新菜单项
</button>
```

### 2. 样式

修改下拉菜单的暗色/亮色主题：

```vue
<!-- 在 UserProfile.vue 中修改 class -->
class="absolute right-0 mt-2 w-48 bg-white rounded-lg shadow-lg"
```

### 3. 头像大小和样式

```vue
<!-- 修改头像尺寸 -->
class="w-10 h-10 rounded-full"
<!-- 改为 w-12 h-12 等 -->
```
